Our Technology

All of our classrooms feature the most advanced interactive SMART Boards, which can not only function as regular white boards, but can also function in conjunction with the latest computer technology. The boards are also capable of using both features simultaneously, and we have the added benefit of advanced software. We use both Easiteach and Wizteach, both software programs that use advanced, interactive teaching tools to engage students throughout lessons.

All upper elementary students are given keyboarding lessons, lessons on supervised internet research, and computer access/practice in the classroom.

All of our elementary students have subscriptions to iXL, the interactive math practice software that is designed to align perfectly with the math common core standards; students can practice each common core requirement individually, and teachers are able to track student progress, accomplishments, and updates.

All of our elementary students have in-class and at-home online access to the new Georgia Common Core State Standards (GA CCSS) for math and language arts programs. Online access includes games, activities, printable worksheets, tutorials and more!
